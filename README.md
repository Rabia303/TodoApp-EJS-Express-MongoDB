

# EJS-MongoDB-CRUD-App  

## Objective  
A basic CRUD application where users can add, view, and delete tasks.  

### Tech Stack  
- **Frontend:** EJS  
- **Backend:** Express  
- **Database:** MongoDB (with Mongoose)
